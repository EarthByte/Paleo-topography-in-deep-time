Overview
---
This repository contains the main workflow and associated scripts used to generate the box and violin plots presented in the above manuscript.<br>
It is designed to facilitate data re-examination and reproducibility for reviewers and readers.<br>

* Included Data<br>
  * Two simplified datasets are provided for transparency and reproducibility:<br>
    * Global Paleo Crustal Thickness Dataset – used to reconstruct continental thickness evolution through time.<br>
    * Age–Temperature Dataset – interpolated from the mantle potential temperature curve of Herzberg et al. (2010).<br>
* Usage Instructions<br>
  * Download the datasets and main analysis script to your local directory.<br>
  * Update the file paths in the script to match your local environment.<br>
  * Run the analysis in a JupyterLab environment.<br>
* Recommended Environment<br>
  * Python version: 3.9<br>
  * Environment: virtual environment (e.g., venv or conda)<br>
  * Suggested IDE: JupyterLab<br>
