Overview
This repository contains the main workflow and associated scripts used to generate the box and violin plots presented in the above manuscript.
It is designed to facilitate data re-examination and reproducibility for reviewers and readers.
Included Data
Two simplified datasets are provided for transparency and reproducibility:
Global Paleo Crustal Thickness Dataset – used to reconstruct continental thickness evolution through time.
Age–Temperature Dataset – interpolated from the mantle potential temperature curve of Herzberg et al. (2010).
Usage Instructions
Download the datasets and main analysis script to your local directory.
Update the file paths in the script to match your local environment.
Run the analysis in a JupyterLab environment.
Recommended Environment
Python version: 3.9
Environment: virtual environment (e.g., venv or conda)
Suggested IDE: JupyterLab
